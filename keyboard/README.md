# TBK mini (6x3+3) wired and wireless.

Colemak-DH + Better symbols.
**Zmk and Qmk configurations.**

This layout is optimized for a great developer experience with:

- Vim bindings
- and a better symbol mapping.

Perfect for integration with Niri, Kitty, Neovim, Fish...
